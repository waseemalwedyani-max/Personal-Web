# Smart Methods Summer Training 2026 - Personal Portfolio Task

## Project Overview
This website is the final submission for the "Create and design a personal website" task under the **Smart Methods Summer Training 2026** program. The objective was to design a professional digital portfolio to showcase my engineering background, technical skills, and practical projects.

## Implementation Workflow (How I built it)

### 1. Development Environment
- **IDE:** Used **Visual Studio Code** for all coding and file structure management.
- **Version Control:** Employed **Git** to track project history and manage commits.

### 2. Design & Architecture
- **Template Selection:** Chose the "Hyperspace" template by HTML5 UP for its clean, modular structure, which provides a professional layout suitable for an engineering portfolio.
- **Frontend Technologies:** Utilized **HTML5** for page structure, **CSS3** for styling, and **JavaScript** for template interactivity.

### 3. Content Integration
- **Profile Construction:** Structured my profile as an Electrical Engineering student (specializing in Embedded Systems and AI).
- **Projects Categorization:**
    - **GitHub Projects:** Integrated links to my software-based projects (e.g., AI Image Classification, Robot Design).
    - **Engineering & Research:** Documented hardware experiments, including the Automatic Power Factor Correction (APFC) system and power electronics analysis.
- **Customization:** Tailored the template by modifying the `index.html` to integrate my specific project data, personal contact information, and social links.

### 4. Deployment Pipeline
- **Hosting:** Utilized **GitHub Pages** for continuous hosting.
- **Workflow:** 
    1. Configured the GitHub Repository.
    2. Pushed the full project directory (including `assets/` and `images/` folders) to the `main` branch.
    3. Enabled "GitHub Pages" in the repository settings to render the website publicly.
