# 🚀 Personal Portfolio Website - Project Documentation

## 📝 Project Overview
This project constitutes the final submission for the assigned task under the Smart Methods Summer Training 2026 program. The primary objective was to design and deploy a professional personal portfolio website to showcase engineering projects, research interests, and technical skills in embedded systems and AI.

---

## 🛠️ Technical Implementation Process

A structured engineering approach was followed to execute this task:

### 1. Planning & Design Phase
*   **Framework Selection:** The "Hyperspace" template was selected for its clean, modular, and responsive design, which provides a professional interface for an engineering portfolio.
*   **Goal Definition:** The focus was to highlight expertise in Power Electronics, Smart Grids, and Embedded Systems.

### 2. Customization & Integration
*   **Content Integration:** `index.html` was customized to map specific data, including a career objective for an Electrical Engineering student and categorized project sections.
*   **Styling:** CSS adjustments were applied to ensure a modern, academic, and professional aesthetic.

### 3. Deployment Pipeline
*   **Version Control:** Git was utilized to manage local file history and track changes.
*   **Hosting:** The project was deployed via GitHub Pages, allowing for continuous hosting directly from the repository.

---

## ⚙️ Engineering Projects Featured

Key engineering milestones highlighted in the portfolio include:
*   **🤖 Automatic Power Factor Correction (APFC) System:** Designed a hardware prototype using Raspberry Pi Pico 2 and PZEM-004T sensors.
*   **🧠 AI Image Classification:** Developed an AI-based image recognition project using Python.
*   **🔧 Quadruped Robot:** Performed mechanical design and torque analysis.
